#### Inspiration

Some ideas for agents to get you started.

- **Automate messy internal work:** Agents that handle real, brittle workflows like support triage, sales ops, compliance checks, data cleanup, or incident response .
- **Build tool-driven agents:** Agents that actively use APIs, SDKs, and integrations, and choose the right tool at the right time instead of relying on prompts alone.
- **Narrow agents** built for one domain (healthcare, fintech, legal, logistics, marketing, dev tools) that solve a clear, real problem.
- **Show measurable impact:** Agents that clearly demonstrate time saved, steps removed, or errors reduced, even with rough numbers.
- **Multi agents:** Applications where different agents plan, execute, review, and verify work, including resolving disagreements.
- **Embed agents where work already happens:** Agents that live inside Slack, email, IDEs, ticketing tools, or internal dashboards and feel native to existing workflows.
- **Connect disconnected systems:** Agents that glue together tools that don’t normally integrate, like CRM + support, GitHub + CI, or calendar + email.
- **Time-series and geo aware agents:** Agents that detect patterns across time and location. Spotting anomalies, trends, or events in metrics, logs, or sensor data, and taking action based on when and where something changed.
- **Let agents take reliable action:** Agents that safely execute decisions creating tickets, deploying fixes, updating records—and explain what they did and why.
